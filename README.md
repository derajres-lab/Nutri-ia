# 🍎 NutriSnap AI
Escáner nutricional con IA — Detección múltiple con YOLOv8 + Open Food Facts

NutriSnap AI permite:
- Detectar varios alimentos a la vez con YOLOv8
- Obtener información nutricional desde Open Food Facts
- Registrar tu consumo diario
- Funcionar 100% local sin enviar imágenes a servidores externos

## 🚀 Instalación local
```bash
pip install -r requirements.txt
streamlit run app.py
